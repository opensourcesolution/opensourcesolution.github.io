OTRS - Helpdesk Open Source

OTRS (Open Technology Real Services) é uma ferramenta de gestão de Help Desk. Seu código é aberto, estável e altamente flexível. Trata-se de uma aplicação Web desenvolvida em linguagem Perl, altamente escalável e flexível. Pode ser instalada em sistemas operacionais Linux, Windows e Unix em conjunto a um servidor web e é acessado pela maioria dos navegadores modernos. As funcionalidades básicas podem ser verificadas no link https://www.otrs.com/feature-overview-of-our-help-desk-solution/ , destacando-se a possibilidade para o uso em smartphones e tablets.

Para acessarmos o instalador web, abra o navegador e acesse o link abaixo: (troque o IP abaixo pelo do seu servidor):

http://ipdoseuservidor/otrs/installer.pl

A tela instalação do OTRS acessada pelo navegador é apresentada abaixo, clique no botão “Próximo” para continuar.

Aceite os termos da licença clicando no botão “Aceitar licença e continue”

Selecione o tipo de Banco de Dados e se deseja criar ou usar um banco de dados existente.

Informe o usuário do banco de dados e sua senha, em seguida clique no botão “Verificar configurações do banco de dados”.

Após as verificações de configurações do banco de dados, o sistema habilita novos campos que servem para a criação de um novo usuário e uma nova base de dados com o nome de “otrs”. O sistema gera uma senha de nível complexo para esse novo usuário, mas aceita uma senha definida por você.

Nesta etapa informaremos:

FQDN(Fully Qualified Domain Name): debian.local.net
E-mail de administrador: suporte@opensourcesolution.com.br
Organização: Open Source Solution
Módulo Registro: Syslog
Idioma padrão: Português
Verificar registro MX: Não

Neste post não configuraremos o envio e recebimento de e-mail,  ficará para um próximo post. Então clique no botão “Pular este passo”.

Na finalização da instalação é apresentada o link de acesso ao sistema, junto com o nome de  usuário e uma senha gerada pelo sistema, copie a senha em um local que possa ter acesso.

Finalizando a instalação
